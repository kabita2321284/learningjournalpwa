My Projects

Here, you’ll showcase your creative computing and technical projects as the course progresses. Each project reflects specific skills and topics from the Mobile App Development module.

  • Week 1: Introduction to Progressive Web Apps and project setup.
  • Week 2 – Learning Journal PWA: Built a responsive website using HTML and CSS to document weekly progress.
  • Week 3 – JavaScript and DOM Manipulation: Added dynamic navigation, collapsible sections, and interactivity using JavaScript.
  • Week 4 – Introduction to APIs: Integrated Storage API (LocalStorage), Clipboard API, and Google Maps Embed API to enhance interactivity and usability within the PWA.
  • Week 5 – Python & JSON Backend Data: Used Python to read and write data to a JSON file, enabling backend-style data storage. Implemented reflection counting and keyword filtering to extend JSON functionality.
  • Week 6 – Frontend–Backend Integration with Flask: Connected the PWA frontend to a Flask backend using GET and POST requests, and implemented DELETE as an extra feature. Reflections are stored in a JSON file on PythonAnywhere, allowing shared, persistent data and full CRUD-style interaction.
  • Week 7 – Progressive Web App: Enhanced the Flask application with PWA features including a service worker and Cache Storage API for offline access and installability. Used Fetch API for dynamic data loading and added an offline status indicator for improved usability.
